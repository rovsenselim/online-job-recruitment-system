# Online Job Recruitment System

## Description
Online Job Recruitment System is a full-stack web application that allows users to manage job recruitment processes. The system enables companies to post job opportunities and applicants to browse and apply for jobs.

This project was built to practice full-stack web development using modern technologies such as React and Node.js.

## Technologies Used
- React
- Node.js
- Express.js
- MongoDB
- JavaScript
- HTML
- CSS

## Features
- User registration and authentication
- Job posting system
- Job application management
- Responsive user interface
- Full-stack architecture
